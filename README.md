# Cartoon Pipeline AUTO v2

Розширена версія cartoon-pipeline з функціями експорту для автоматизації.

## Що нового порівняно з оригіналом

- **Кнопка "Higgsfield промпти (.txt)"** — завантажує пронумерований текстовий файл з усіма image prompts. Монологи позначені окремо.
- **Кнопка "Magnific List (.json)"** — завантажує JSON тільки з НЕ-монологами, готовий для List node в Magnific Spaces.
- **Кнопка "Усі дані (.json)"** — повний дамп всього для архіву або інтеграцій.
- **Бейдж #номер** — кожна сцена має номер, щоб легко розкласти по порядку.
- **Позначка "монолог — окремо"** — сцени з довгими монологами виділені жовтою смугою.
- **Статистика** — показує скільки сцен для Magnific, скільки монологів, загальний хронометраж.

## Deploy to Railway

1. Завантаж цю папку на GitHub (новий репозиторій)
2. Зайди на railway.app → New Project → Deploy from GitHub
3. Вибери репозиторій
4. Variables → додай: `OPENAI_API_KEY` = твій ключ
5. Deploy — Railway дасть нову URL
